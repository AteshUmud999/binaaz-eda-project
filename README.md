🏠 Bina.az Real Estate Data Analysis
📌 Project Overview

This project focuses on collecting, cleaning, and analyzing real estate listings from Bina.az. The main objective is to perform a complete Exploratory Data Analysis (EDA) by applying data preprocessing techniques, feature engineering, visualization, and extracting meaningful insights from the dataset.

The project is completed as part of the Data Science Online Internship program.

🎯 Objectives
Scrape real estate listing data from Bina.az
Clean and preprocess the collected dataset
Handle missing values using justified strategies
Detect and handle outliers
Create meaningful new features
Perform exploratory data analysis (EDA)
Visualize important trends and relationships
Generate data-driven insights
📊 Dataset Information

The dataset contains information about property listings, including:

Property price
Price per square meter
Property category
Location
Address
Number of rooms
Area
Floor information
Property condition
Mortgage availability
Ownership information
Listing views
Geographic coordinates
Listing date and time

📋 Project Workflow
1. Data Collection
Scraped property listings from Bina.az
Built a structured dataset
2. Data Cleaning
Removed duplicate records
Fixed inconsistent values
Converted data types
Standardized numerical values
3. Missing Value Handling
Analyzed missing values
Applied feature-specific strategies

4. Outlier Detection and Handling
Detected outliers using the IQR method
Investigated extreme observations
Removed unrealistic values while preserving meaningful extreme properties
5. Feature Engineering
Created new meaningful features
Improved dataset usability for analysis
6. Data Visualization
Distribution analysis
Correlation analysis
Category-based visualizations
Comparative charts
7. Insights
Identified important trends within the housing market
